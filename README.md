The goal of this project is to build a playable version of chess with code. I will also learn to build some sort of evaluation engine using minimax and alpha beta pruning. 
Ultimately, after I have a working prototype, I want to introduce my own pieces and rulesets and build an engine on top of this as well.

I can then extend this into a full stack application with Java/SpringBoot or just keep everything JavaScript using Node.js and Express. 
The planned features are real time gameplay across devices using WebSockets, account creation and win/loss storage, game history/replay and evaluation on replay.
